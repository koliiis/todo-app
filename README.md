# ToDo App


## Overview

### The challenge

Users should be able to:

  - Sign In or Log In
  - Create, Delete and Update the status of TODO or rename it
  - Filter by All, Active, and Completed
  - Delete all TODOs
  - Toggle status of all TODOs
